# Tasty road

* 맛집을 검색, 등록하여 맛집 지도 위에 공유.<br>
* 위치, 거리, 리뷰, 평점 등 한 눈에 정보 파악.

## 기술
* React.js, html, css
* Node.js, Express.js, MongoDB, Mongoose
